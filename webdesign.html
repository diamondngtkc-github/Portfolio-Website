cycloid (2020). I created a website design and video concept for Cycloid, a tech start-up based in Paris, France. My demo focused on one of their main products - TerraCognita, which was a tool to discover deployed resources in the cloud infrastructure and export them to Terraform configuration files. TerraCognita would allow users to migrate the current cloud infrastructure to infrastructure-as-code.

Medium: 2D design

Role: UI Designer and Storyboarding Artist

Client: Cycloid
